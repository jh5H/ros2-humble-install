# ros2 humble install with script.sh
---

<br/>

#### clone

```bash
cd ~
git clone https://github.com/jh5H/ros2_humble_install.git
```

<br/>

#### change mode execute && install

```bash
cd ~/ros2_humble_install
sudo chmod +x humble_install.sh
./humble_install.sh

```
